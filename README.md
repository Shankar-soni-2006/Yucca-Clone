# Yucca-Clone
# Yucca Packaging Website Clone

A visually rich **frontend clone of the Yucca Packaging website** built using **HTML, CSS, and JavaScript**.
The project focuses on modern UI design, smooth animations, and interactive components inspired by the original Yucca Packaging website.

This project demonstrates advanced frontend concepts such as **hover animations, parallax scrolling, product cards, marquees, and interactive layouts**.

---

## 🚀 Live Preview
https://yucca-clone-xi.vercel.app

---

## 📌 Features

* Modern **responsive landing page design**
* **Smooth scrolling effects** using Locomotive Scroll
* Interactive **hover animations and transitions**
* Animated **product showcase cards**
* Infinite **marquee scrolling sections**
* Custom **FAQ section**
* **Dynamic UI elements** such as:

  * Animated buttons
  * Hover image effects
  * Card expansion animations
* Clean and organized **section-based layout**

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome Icons**
* **Google Fonts**
* **Locomotive Scroll (for smooth scrolling effects)**

## ⚙️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/yourusername/yucca-packaging-clone.git
```

2. Navigate into the project folder

```bash
cd yucca-packaging-clone
```

3. Open `index.html` in your browser.
